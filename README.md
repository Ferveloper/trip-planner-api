# Trip Planner API REST

An API REST built in Node.js/Express and MongoDB to keep track of travel expenses.

## Requirements

- Node.js
- MongoDB

## Installation

```
> git clone https://github.com/Ferveloper/trip-planner-api.git
> cd trip-planner-api
> npm i
> npm start
```
